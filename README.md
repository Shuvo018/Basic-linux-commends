# Basic-linux-commends
| Command   |               Description     |
|-----------|-------------------------------|
| pwd       | Show current directory        |
| ls        | List files/folders             |
| ls -l        |    Detailed file list          |
|  ls -a       |   Show hidden files           |
|   cd      |   Change directory           |
|  cd ..       |   Go back one folder           |
| cd ~          | Go to home directory |
